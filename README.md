# rkumar1990.github.io

This is the personal website and blog of Ranjeet Kumar, a seasoned technical writer with over 11 years of experience. The site showcases technical writing guides, tutorials, and professional information.

## About

The website is built using Jekyll and hosted on GitHub Pages. It features:

- Blog posts on technical writing topics
- Professional resume
- Guides for aspiring technical writers

## Local Development

To run this site locally:

1. Ensure you have Ruby and Bundler installed. Follow the [Jekyll installation guide](https://jekyllrb.com/docs/installation/) and install Bundler if needed (`gem install bundler`).

2. Clone the repository:
   ```bash
   git clone https://github.com/rkumar1990/rkumar1990.github.io.git
   cd rkumar1990.github.io
   ```

3. Install dependencies:
   ```bash
   bundle install
   ```

4. Serve the site locally:
   ```bash
   bundle exec jekyll serve
   ```

5. Open your browser to `http://localhost:4000` to view the site.
